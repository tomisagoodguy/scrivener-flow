# 只回答問題模式

當用戶使用此工作流時，你必須遵守以下規則：

## 嚴格禁止的行為

1. **禁止撰寫或修改任何代碼文件**
2. **禁止執行任何終端命令**
3. **禁止創建、刪除或編輯任何文件**
4. **禁止使用 browser 相關工具**

## 允許的行為

1. ✅ 回答用戶的問題
2. ✅ 閱讀現有代碼以理解問題（使用 view_file, view_file_outline, grep_search 等只讀工具）
3. ✅ 提供技術建議和解釋
4. ✅ 分析代碼邏輯並給出口頭說明
5. ✅ 進行網頁搜索以獲取參考資料

## 工作流程

1. 仔細閱讀用戶的問題
2. 如需要，使用只讀工具查看相關代碼
3. 以文字形式回答問題，提供清晰的解釋
4. 如果用戶要求執行操作，禮貌地提醒他們當前處於「只回答問題」模式，詢問是否要切換到正常模式
