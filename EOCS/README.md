# 📚 Engineering Operations & Code Standards (EOCS)

此目錄是專案唯一的技術文件存放中心 (Single Source of Truth)。
所有前後端文件、API 規格、資料庫 Schema 說明都應存放於此，並依功能分類。

## 📂 目錄結構

- `frontend/`: 前端架構、UI 元件規範
- `backend/`: 後端架構、API 說明
- `schema/`: 資料庫 Schema、資料流
- `ops/`: 部署、CI/CD、運維手冊

## 📝 規範

所有文件必須：

1. 使用繁體中文撰寫
2. 定期更新以保持與程式碼同步
3. 遵循 Markdown 格式
